本项目实现一个 key - value 存储项目
一、写操作
    1. 先写 wal 日志，格式为：crc32(4) + type(1) + KeyLen(4) + ValLen(4) + Key + Value

    2. 内存层: 写 MemTable 使用跳表数据结构

    3. SSTable 层：当 MemTable 满时，将 MemTable 的所有数据持久化到磁盘中，格式为：
        record1: Len(4) + [KeyLen(4) + Key + ValLen(4) + Value]
        record2: Len(4) + [KeyLen(4) + Key + ValLen(4) + Value]
        record3: Len(4) + [KeyLen(4) + Key + ValLen(4) + Value]
        ...

        会记录 keys 和 offsets，用来索引查找


    4. 磁盘中的文件变多时，使用 Compaction 压缩

二、读操作
    1. 先读 MemTable
    2. 读 SSTable

三、删除操作
    1. 写删除标记到 WAL 日志
    2. 写删除标记到 MemTable

四、恢复操作
    从 WAL 日志读取操作记录，解析出 key 和 value
    然后将这些 key value 写入 MemTable
